# 98przem/homebrew-tap

Homebrew tap for [Mist](https://github.com/98przem/mist) — play Windows Steam and Epic games on macOS via Wine.

## Install

```sh
brew install --cask 98przem/tap/mist
```

## Uninstall (with all data)

```sh
brew uninstall --zap mist
```

The `--zap` flag removes Mist itself plus its entire Wine prefix, downloaded games,
engine, logins, and preferences.
